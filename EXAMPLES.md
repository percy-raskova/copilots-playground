## Notes on Execution

Remember: **Time flows backwards!** The last statement in the source code executes first. This means:

1. You declare variables before you use them (but write them after in source)
2. Functions must be defined before they are called (but appear after in source)
3. Output appears in reverse order of source code
4. Loops count down by default (it's more natural when time reverses)
